Important things

URL - Uniform Resource Locator

A URL, colloquially termed a web address,[1] is a reference to a web resource that specifies its location on a computer network and a mechanism for retrieving it.

Most web browsers display the URL of a web page above the page in an address bar. A typical URL could have the form http://www.example.com/index.html, which indicates a protocol (http), a hostname (www.example.com), and a file name (index.html).

Web Server

A web server is server software, or hardware dedicated to running this software, that can satisfy client requests on the World Wide Web. It can contain one or more websites

Function to store, process and deliver web pages to clients

Web Browser

A web browser is a software application for accessing information on the World Wide Web.

The purpose is to fetch information resources from the Web and display them on a user's interface

The function: This process begins when the user inputs a Uniform Resource Locator (URL), such as https://en.wikipedia.org/, into the browser. Most browsers use an internal cache of web page resources to improve loading times for subsequent visits to the same page.

Identify attributes

->All HTML elemts can have attributes

->Attributes provide additional information about elements

->Attributes are always specified in the start tag

->Attributes usually come in name/value pairs like: name="value"

The href Attribute: The < a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

The src Attribute:The < img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed:< img src="img_girl.jpg">

There are two ways to specify the URL in the src attribute:1. Absolute URL - Links to an external image that is hosted on another website. Example: src= "https://www.w3schools.com/images/img_girl.jpg".

2. Relative URL - Links to an image that is hosted within the website. Here, the URL does not include the domain name. If the URL begins without a slash, it will be relative to the current page. Example: src="img_girl.jpg". If the URL begins with a slash, it will be relative to the domain. Example: src="/images/img_girl.jpg".

Tip: It is almost always best to use relative URLs. They will not break if you change domain.

The width and height Attributes:The < img> tag should also contain the width and height attributes, which specifies the width and height of the image (in pixels):< img src="img_girl.jpg" width="500" height="600">

The alt Attribute:The required alt attribute for the tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to slow connection, or an error in the src attribute, or if the user uses a screen reader.(< img src="img_girl.jpg" alt="Girl with a jacket")>

The style Attribute:The style attribute is used to add styles to an element, such as color, font, size, and more.(< p style="color:red;">This is a red paragraph.< /p>)

The lang Attribute:You should always include the lang attribute inside the < html> tag, to declare the language of the Web page. This is meant to assist search engines and browsers.

The title Attribute:The title attribute defines some extra information about an element.(< p title="I'm a tooltip">This is a paragraph.

Name the components of a css rule properly

A CSS rule-set consists of a selector and a declaration block.

The selector points want to style (Example: < h1 >)

A declaration block contains one or more declarations (seperated by semicolons). Each declaration includes a CSS property name and a value, separated by a colon.

Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

Use combinators properly: direct child, descendant, adjacent sibling, sibling (??)

Declarations for color, background color

Declaration Color: Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.

Declaration Backgroundcolor: The CSS background properties are used to define the background effects for elements. (background-color: lightblue;opacity: 0.3;)

The background-image property specifies an image to use as the background of an element. By default, the image is repeated so it covers the entire element. The background-image property repeats an image both horizontally and vertically.

The background-position property is used to specify the position of the background image.

Pseudo classes for the < a > element

A pseudo-class is used to define a special state of an element.

It can be used to: Style an element when a user mouses over it, style visited and unvisited links differently, style an element when it gets focus

a:link {color: #FF0000;} (for an unused link ex. blue, for used link ex. purple)

Declaration for background image

The background-image property specifies an image to use as the background of an element. (in Body {} background-image: url("paper.gif");)

Opacity: specifies the opacity/transparency of an element. It can take a value from 0.0 - 1.0.

Declaration for text alignment (left, right, center, justify)

The text-align property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified. (in h {} text-align: center;)

Declaration for text decoration (overline, underline, line-through)

The text-decoration property is used to set or remove decorations from text. The value text-decoration: none; is often used to remove underlines from links. (in a {} text-decoration: underline;)

Declaration for text transformation (uppercase, lowercase, capitalize)

The text-transform property is used to specify uppercase and lowercase letters in a text. It can be used to turn everything into uppercase or lowercase letters, or capitalize the first letter of each word (in p.uppercase {} text-transform: uppercase;)

Identify the difference between serif and sans-serif fonts

Serif: Serif fonts have small lines at the ends on some characters (.serif)

Sans-serif: "Sans" means without - these fonts do not have the lines at the ends of characters (.sansserif)

Declarations for font families

The font-family property should hold several font names as a "fallback" system. Start with the font you want, and end with a generic family, to let the browser pick a similar font in the generic family, if no other fonts are available. (font-family: Arial, Helvetica, sans-serif;)

Declarations for font style normal and italic

The font-style property is mostly used to specify italic text.

normal - The text is shown normally || italic - The text is shown in italics

Declarations for font sizes

The font-size property sets the size of the text. (h1 {font-size: 20px;})

Absolute size: Sets the text to a specified size | Does not allow a user to change the text size in all browsers (bad for accessibility reasons) | Absolute size is useful when the physical size of the output is known

Relative size: Relative size: Sets the size relative to surrounding elements | Allows a user to change the text size in browsers

Declarations for font weights

The font-weight property sets how thick or thin characters in text should be displayed. (p.normal {font-weight: normal;})
